# Talent Restart | Kariéra pro sportovce

První pracovní portál v České republice propojující sportovní talent s reálnými pracovními příležitostmi.

## Project info

Tento projekt je postaven na moderním stacku:
- **Vite** (Build tool)
- **React** (UI library)
- **TypeScript** (Language)
- **Tailwind CSS** (Styling)
- **Shadcn/UI** (Components)

## Getting Started

1.  **Instalace závislostí**
    ```sh
    npm install
    ```

2.  **Spuštění lokálně**
    ```sh
    npm run dev
    ```

3.  **Build pro produkci**
    ```sh
    npm run build
    ```

## Pro sportovce
Pomáháme sportovcům najít novou kariéru po skončení profesionálního sportu.

## Pro zaměstnavatele
Umožňujeme firmám najít disciplinované a vytrvalé talentované lidi se sportovní minulostí.
